# G2I Shortcut

This is a short alias for `$gpt-image-2-prompts`.

When invoked, use the main skill instructions from:

```text
~/.codex/skills/gpt-image-2-prompts/SKILL.md
```

If that local path is unavailable, use the same operating model:

- Write GPT-Image-2 prompts as visual direction, not keyword piles.
- For reusable requests, create a compact Prompt-as-Code schema first.
- For series, define a shared Style Lock before per-image prompts.
- Include exact text, layout, composition, lighting, materials, and constraints.
- Avoid malformed text, random logos, inconsistent identity, clutter, wrong panel counts, and unreadable labels.
